************
User Guide
************

Welcome to the User Guide for the Python Code Validator. This guide is
designed for end-users, such as teachers and curriculum developers, who want
to use the validator to create and run checks on Python code.

Here you will find everything you need to get started, from installation to
writing your first validation rules.
